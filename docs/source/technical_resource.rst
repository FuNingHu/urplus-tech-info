Frequently Asked Questions
--------------------------

**Q: How do I check my PolyScope version?**

A: Go to **Settings** > **System** > **About** to view the PolyScope version.

**Q: Can I install multiple URCaps?**

A: Yes, multiple URCaps can be installed simultaneously, provided there are no conflicts.

**Q: How do I update a URCap?**

A: Remove the existing URCap, restart, then install the new version.

Reporting Issues
----------------

To report a bug or issue:

1. Gather relevant log files from the robot
2. Document steps to reproduce the issue
3. Submit a detailed report via email or support portal

.. tip::
   Include screenshots or videos when reporting visual issues.
